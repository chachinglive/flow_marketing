FLOW: Tự Động Theo Dõi Kênh YouTube “VTV24” (theo_doi_kenh_y_report.txt)
Mục tiêu: Mở YouTube → tìm “vtv24” → tự động nhấn nút ĐĂNG KÝ kênh chính thức.

Bước 1: Chạy FLOW: Gmail Login 2FA(gmail_login_2fa) để đăng nhập trước khi tìm kiếm
Bước 2: Chạy FLOW: watch_video_youtube

NODE 1.
# 1. Mở YouTube
self.lib_ToUrl('https://www.youtube.com')
-> Vào trang chủ YouTube

# 2. Tìm kênh muốn theo dõi
a = self.lib_ElGetXPath('//yt-searchbox[@role="search" and contains(@class, "ytSearchboxComponentHost")]')
-> Tìm đúng khung search có tag <yt-searchbox> và class chứa “ytSearchboxComponentHost”

if a is not None:
  a_children = self.lib_ElGetXPathFrom(a, './/input[@role="combobox" and @name="search_query"]')
  -> Lấy ô input có name="search_query"

  if a_children is not None:
    self.lib_ElSendKeys(a_children, 'vtv24')
    self.lib_SendKeysEnter(a_children)
    -> Gõ “vtv24” + Enter → ra danh sách kênh/video


NODE 2 – Nhấn ĐĂNG KÝ (giữ nguyên 100% code gốc)
# Nhấn đăng ký
self.lib_Sleep(15)
-> Chờ 15 giây để trang kết quả / trang kênh load xong

a = self.lib_ElGetXPath('//div[@id="subscribe-button" and contains(@class, "style-scope") and contains(@class, "ytd-channel-renderer")]')
-> Tìm vùng nút subscribe có id="subscribe-button"

if a is not None:
  a_children = self.lib_ElGetXPathFrom(a, './/button[contains(@aria-label, "Đăng ký") and contains(@class, "yt-spec-button-shape-next")]')
  -> Tìm đúng button có aria-label chứa “Đăng ký” và class “yt-spec-button-shape-next”

  self.lib_ElClick(a_children)
  -> Click → kênh VTV24 được theo dõi ngay lập tức
